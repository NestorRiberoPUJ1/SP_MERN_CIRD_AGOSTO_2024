ETAPA DE PROYECTOS

GRUPOS

  • Libre escogencia
  • Grupos de 4 estudiantes
  • Me deben enviar un correo con el listado del grupo
  • Quienes no envíen correo antes del próximo martes o hayan sido incluidos en algún grupo los asisgnaré yo

PROYECTO

  • Tema es de libre escogencia
  • Sugerido no E-commerce porque requiere mucho tiempo de dedicación
  • Definir claramente que función tiene o que problema soluciona
  • Debe tener alguna forma de sustento económico (suscripciones,donaciones,comiciones, etc...)
  • No es necesario implementar las funcionalidades de monetización
  • Debe serguir la estructura de MERN
    • Base de datos en MONGODB
    • Backend en Express
    • Frontend en REACT
    • Todo en NodeJS
  • Finalmente se debe hacer una presentación del proyecto

Entregables

Se espera que para la finalización del proyecto se tenga un Producto Minimo Viable (MVP) que incluya las funcionalidades esenciales del proyecto definido por el grupo

  • Descripción del proyecto
  • Distribución de roles (cómo van a trabajar)
  • Calendario de avances (qué esperan tener en qué día)
  • Alcance del proyecto (hasta qué funcionalidades esperan llegar)
  • Presentación del proyecto
